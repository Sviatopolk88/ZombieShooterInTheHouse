# EnemyAnimationController

## Назначение
Связывает состояние врага с Animator.

## Компоненты
- EnemyAnimationController — управляет параметрами Animator

## Использование
Добавить на корневой объект врага. `Animator` может находиться на дочернем объекте

## Интеграция
Используется EnemyAI_Base для вызова атак, получения урона и смерти

## Примечания
- Animator должен быть на дочернем объекте
- Root Motion отключён
- Компонент не подписывается на здоровье самостоятельно
