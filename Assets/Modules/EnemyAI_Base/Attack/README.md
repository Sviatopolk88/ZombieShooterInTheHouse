# EnemyAttack

## Назначение
Наносит урон по задержке после старта атаки

## Компоненты
- `EnemyAttack` — ждёт `attackDelay`, проверяет дистанцию и наносит урон цели
- `AttackHitbox` — legacy-компонент контактного урона, по умолчанию отключён

## Использование
- В момент атаки вызвать `EnemyAttack.StartAttack(target)`
- `EnemyAnimationController` отдельно запускает анимацию атаки

## Примечания
- `EnemyAttack` не управляет движением
- `EnemyAttack` не запускает анимацию
- окончание атаки определяется по состоянию `Animator`
- `attackDelay` должен совпадать с моментом удара в клипе
- если `attackDelay` больше длины анимации, урон не будет нанесён
